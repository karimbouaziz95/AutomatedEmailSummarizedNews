# NewsAPIEmail

A simple application that fetches news articles from NewsAPI and sends them via email.

## Features
- Fetch latest news articles by topic
- Send curated news via email
- Configurable news sources and categories

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set your NewsAPI key and email credentials in `.env`
4. Run the application: `python main.py`

## Requirements
- Python 3.8+
- NewsAPI key
- Email account for sending
